## 🔍 현재 파악된 정보

### 백엔드 API 구조
- **기본 URL**: `http://localhost:8080`
- **인증**: JWT 토큰 기반
- **응답 형식**: CustomReturnDTO (message, status, statusCode, data)

### 주요 API 엔드포인트
- **로그인**: `POST /api/kitms-login`
- **토큰 갱신**: `GET /api/kitms-login/refresh`
- **사용자 ID 찾기**: `GET /api/kitms-login/id`
- **비밀번호 재설정**: `GET /api/kitms-login/pwd`
- **로그아웃**: `GET /api/kitms-login/logout`
- **토큰 유효성 검사**: `GET /api/kitms-login/valid-token`
- **공지사항 목록**: `GET /api/kitms-notices/all`
- **공지사항 상세**: `GET /api/kitms-notices/{noticeNo}`
- **공지사항 등록**: `POST /api/kitms-notices` (multipart)
- **공지사항 수정**: `PUT /api/kitms-notices` (multipart)
- **공지사항 삭제**: `DELETE /api/kitms-notices/{noticeNo}`
- **파일 다운로드**: `GET /api/kitms-attaches/{attachNo}/download`
- **파일 ZIP 다운로드**: `GET /api/kitms-attaches/{attachNo}/download-zip`
- **로그인 로그**: `GET /api/kitms-login-logs`

### 데이터베이스
- **타입**: MariaDB
- **개발용 URL**: `jdbc:mariadb://localhost:3306/testsh`
- **인코딩**: UTF-8 (utf8mb4)

### 파일 저장
- **경로**: `/var/kitms/upload/`
- **지원 형식**: 모든 파일 형식

### 사용자 권한 시스템
- **관리자**: "0" - 전체 관리 권한
- **AS총괄**: "1" - AS 업무 총괄
- **승인권자**: "2" - 승인 권한
- **AS기사**: "3" - AS 업무 수행
- **고객/요청자**: "4" - 서비스 요청
- **콜센터**: "5" - 고객 상담
- **뷰어**: "6" - 조회 전용
- **익명**: "9" - 제한적 접근

### 페이징 및 검색 파라미터
- **페이징**: page, size, start, end
- **정렬**: sortCol, order
- **검색**: searchColumn, searchValue, searchArray
- **조직**: organNo, organNoList
- **날짜**: searchStartDt, searchEndDt
